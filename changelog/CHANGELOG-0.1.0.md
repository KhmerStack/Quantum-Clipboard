## Overview
Initial public release of Quantum Clipboard.  
This version focuses on core clipboard history functionality with a minimal and stable UI.

## Added
- Clipboard history for text and images
- macOS native popup window
- Global keyboard shortcut to open clipboard
- Basic search functionality
- Clipboard item selection and copy

## Notes
- No theme customization
- No language selection
- UI is fixed and minimal by design
- Local-only storage

## Known Limitations
- No color or appearance settings
- No localization support
- No pinning or tagging
